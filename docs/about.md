# About This Project

This project simulates a real-world partner integration system for ingesting and processing product catalog data.

---

## My Role

- Designed API endpoints and workflows
- Implemented data ingestion and validation logic
- Modeled request and response schemas
- Authored all developer documentation
- Designed documentation structure and navigation
- Deployed application using Docker and AWS

---

## Key Design Concepts

### Data Ingestion Pipeline
- CSV upload via multipart/form-data
- Data normalization and validation
- Persistence to relational database

### Asynchronous Processing
- Submission and validation jobs
- Status tracking via /jobs endpoint

### API Design
- Resource-based endpoints
- Cursor-based pagination
- Filtering and sorting
- Consistent JSON response models
